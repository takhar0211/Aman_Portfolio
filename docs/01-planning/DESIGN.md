# DESIGN

## Design Philosophy

The portfolio should feel modern, premium, minimal, and professional.

The goal is to impress recruiters through clean design rather than excessive animations or visual effects.

Every section should have a clear purpose.

The interface should prioritize readability, spacing, and simplicity.

---

# Overall Design Style

Keywords

- Modern
- Minimal
- Premium
- Elegant
- Professional
- Developer Portfolio
- Clean
- Responsive

Avoid flashy designs.

The website should still look relevant several years from now.

What do I do?

Why should someone continue scrolling?

Should contain

- Name
- Title
- Short introduction
- Primary CTA
- Secondary CTA
- Social Links

Hero should occupy most of the first screen.

# Icons

Avoid decorative icon overload.

---

---

# Images

Use optimized Next.js Image component.

Images should be high quality.

Avoid stock photos.

Prefer screenshots of actual projects.

---

# Responsive Design

The website must be fully responsive.

Layouts should adapt smoothly.

No horizontal scrolling.

Cards should stack naturally.

Navigation should collapse into a hamburger menu.

Buttons should remain easy to tap.

---

# Spacing

Use consistent spacing throughout the project.

Prefer whitespace over separators.

Never place components too close together.

Visual hierarchy should remain obvious.

---

# Footer

Simple.

Minimal.

Should include

Name

Copyright

Social Links

Built with Next.js

---

# Accessibility

Maintain sufficient color contrast.

Buttons should have visible focus states.

Links should be clearly distinguishable.

Interactive elements should be keyboard accessible.

---

# Performance

Animations should not reduce performance.

Avoid unnecessary re-renders.

Avoid unnecessary libraries.

Prefer lightweight solutions.

---

# Design Principles

Every UI decision should follow these priorities.

1. Simplicity
2. Readability
3. Consistency
4. Responsiveness
5. Accessibility
6. Performance
7. Maintainability

If multiple design options exist, choose the simpler one.

---

# Inspiration

The overall feel should resemble modern developer portfolios seen on platforms like Vercel, Linear, Stripe, and well-crafted personal portfolio websites.

Do not copy any specific website.

Use them only as inspiration for spacing, typography, and overall design quality.
